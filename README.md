# Multi LLM Evaluation System

![multi-llm-system](header.jpg)

The idea of this project is to evaluate multiple Large Language Models (LLMs) simultaneously instead of relying on a single provider (e.g., OpenAI GPT, Google Gemini, Anthropic Claude, xAI Grok, etc.). 

This is a local web application that provides a ChatGPT-like interface, but internally sends a query to multiple LLMs using OpenRouter. The system then compares, evaluates, and synthesizes their responses to generate a more refined final output.

## How It Works

When a user submits a query, the system follows a structured multi-stage pipeline:

### 1. Stage 1: Initial Responses
The user query is sent to all configured LLMs independently. Each model generates its own response.  
These responses are displayed in a tab-based interface so users can compare outputs side by side.

### 2. Stage 2: Cross Evaluation
Each LLM is provided with responses generated by other models.  
To ensure fairness, model identities are anonymized.  
Each model evaluates and ranks the responses based on:
- Accuracy  
- Completeness  
- Insight  

### 3. Stage 3: Final Synthesis
A designated “Chairman” model analyzes all responses and rankings.  
It then generates a final consolidated answer that combines the best aspects of all outputs.
